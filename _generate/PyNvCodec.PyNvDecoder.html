Overloaded function.

  1. FlushSingleSurface(self: PyNvCodec._PyNvCodec.PyNvDecoder) -> Tuple[VPF::Surface, PyNvCodec._PyNvCodec.TaskExecInfo]

    Send null input to decoder. Use this function in the end of decoding session to flush decoder and obtain those video frames which were not returned yet.

    If this method returns empty Surface it means there are no decoded frames left.

    Video frame is returned as Surface stored in vRAM. :return: tuple, first element is the surface, second is TaskExecInfo.

  2. FlushSingleSurface(self: PyNvCodec._PyNvCodec.PyNvDecoder, pkt_data: PyNvCodec._PyNvCodec.PacketData) -> Tuple[VPF::Surface, PyNvCodec._PyNvCodec.TaskExecInfo]

    Send null input to decoder. Use this function in the end of decoding session to flush decoder and obtain those video frames which were not returned yet.

    If this method returns empty Surface it means there are no decoded frames left.

    Video frame is returned as Surface stored in vRAM.

    param pkt_data:

    PacketData structure of decoded frame with PTS, DTS etc.

    return:

    tuple, first element is the surface, second is TaskExecInfo.
